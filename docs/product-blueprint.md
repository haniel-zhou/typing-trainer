# Typing Trainer 产品蓝图

## 1. 完整功能结构图
- 核心训练：课程关卡、个性化词库、实时纠错、节奏音效、专项训练
- 效率与智能：数据分析、云同步、多端适配、后续联想补全
- 社交与竞技：好友邀请、好友主页、排行榜、好友对战、赛季挑战
- 辅助与无障碍：护眼主题、夜间主题、语音辅助、键盘节奏引导

## 2. 页面结构
- `/` 首页：今日目标、签到、成就墙、课程路线、产品入口
- `/lessons` 课程页：完整关卡路线图
- `/trainer` 练习页：目标文本、输入区、实时数据、错误分析、节奏音乐
- `/custom` 词库页：专业词库、自定义术语、语音录入
- `/challenge` 挑战页：好友榜、全球榜、挑战卡
- `/friends` 好友页：邀请链接、好友列表、好友主页入口
- `/stats` 成长页：速度曲线、热力图、错误热键、练习历史
- `/product` 产品页：功能结构、页面结构、数据库、成长系统、游戏化机制

## 3. 数据库设计
- `profiles`：用户主页与成长汇总
- `training_records`：每轮训练结果、错误分析
- `friends`：好友关系
- `duels` / `duel_attempts`：好友对战
- `word_banks`：词库云同步
- `check_ins`：每日签到
- `achievement_unlocks`：成就解锁

## 4. 用户成长系统
- 单次练习闭环：实时反馈 -> 结算 -> 纠错建议
- 每日习惯闭环：签到 -> 任务 -> 连续训练
- 长期成长闭环：等级 -> 成就 -> 榜单 -> 赛季

## 5. 游戏化机制
- 连击流：高连击触发更强反馈
- 每日任务：引导用户每天都有清晰目标
- Boss 挑战：困难文本与专项关卡
- 好友对战：分享链接即可开打
